# LIBRARY MANAGE SYSTEM

This project simulates a library system designed for librarians or library staff who need to check book borrowings.

## ✨Features
- Add Book (Name,IdBook(Auto Fill the ID),Category)
- ShowBook (Name,IdBook,isAvailable,Genre)
- Search Book
- Borrow Book 
- Return Book
